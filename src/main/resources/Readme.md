##  SISTEMA EVENTO

- Sistema de gerenciamento de informações de participantes de atividades de um evento acadêmico, visando a implementação de modelo de dominio e ORM. 

## Tecnologias utilizadas:
- JAVA - Versão 17;
- Springboot - Versão 3.1.0;
- IDE - Intellij
- H2 Database

## Considerações finais

Dados serão persistidos em um banco de dados em memória.
